ie-hover by weepy

Bringing :hover to IE Stylesheets with jQuery

This very lightweight jQuery plugin adds in support for IE :hover selectors in CSS

Use

Include the script and after your last stylesheet, use the following javascript $.fixIEHover()

Better to avoid a document.ready event here as it's unnecessary. It works by looping through the stylesheet rules, looking for :hover classes and adding equivalent IE friendly classes along with events for mouse in/out with jQuery
